課程教材

本系課程的目標不僅在於基本語文能力的訓練以及專業知識的培養，同時著重人文素養的造就與養成，期能培育出能夠清晰思辨、明確表達、清楚溝通，並且具備歷 史觀與國際觀的二十一世紀文化人。

Learn More

業師分享

課程的安排可粗分為以下各類。「基礎語言能力課程」旨在訓練與加強學生英語聽、說、讀、寫能力，採取小班教學。「文學課 程」及「語言學課程」著重專業知識的培養。

Learn More

學生作品

「第二外語課程」提供了英語以外的外語學習機會，計 有日文、法文、德文、西班牙文供學生選修。此外，尚有翻譯、應用英文、英語教學等眾多其他課程，學生可依興趣及需要選課。

期末發表

學士班課程規劃，課程宗旨與結構，學士班畢業學分為136學分，包括全校共同必修科目30學分﹝大一國文4學分、大一英文6學分及通識教育20學分﹞，專業必修54學分。
